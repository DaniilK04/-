# Habit Tracker 📝

Pet-проект на **Django** для отслеживания привычек и задач.  
Проект демонстрирует работу с Django, формами, аутентификацией пользователей и базовыми возможностями Git/GitHub.

## 🚀 Возможности
- Регистрация и авторизация пользователей
- Создание, редактирование и удаление привычек
- Отметка выполнения привычек
- Поиск и сортировка по параметрам
- Административная панель Django
- Обработка ошибок (403 / 404 / 500)

## 🛠 Стек
- Python 	3.13.5
- Django 	6.0.1
- SQLite (на этапе разработки)
- HTML, CSS
